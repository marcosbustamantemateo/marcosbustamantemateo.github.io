# Personal Website

This is the source code of my personal website/portfolio.

## Local development

Prerequisites:

- Node.js and npm

Steps:


```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build

```sh
npm run build
```
